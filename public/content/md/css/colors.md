# Overview

Use CSS to add color to a site. CSS provides a few ways to do this:

- red green blue alpha (rgba)
- hue saturation light alpha (hsla),
- hexidecimal (#BADA55)
- name (YellowGreen)

## Activities

- Add color to a web page using an example of one of the above ways to add color.
